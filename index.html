Computer networking

Computer networking refers to interconnected computing devices that can exchange data and share resources with each other using communication protocols. .

Web servers

A web server is a computer that stores web server software and a website's component files and it also includes several parts that control how web users access hosted files.

Frameworks

A framework is a guide that you use to structure your code, logic and other aspects of your web architecture. .

Databases

Databases contain the information that servers access to direct website functions. .

APIs

APIs, or application programming interfaces, allow software programs to communicate with other servers and databases to exchange information.

Deployment

Once you finish writing the code and organizing the files that make up your website, you need to put it all online so people can find it. That's what deploying a website means.
